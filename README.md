# S I L O Furniture — Premium E-Commerce Website

A modern, minimalist e-commerce website for premium handcrafted furniture based in Dhaka, Bangladesh.

## Features

- 🏠 Beautiful hero section with luxury living room imagery
- ✨ Value propositions highlighting craftsmanship
- 🛋️ Featured collections gallery
- 🔨 Brand story section
- 📝 Custom furniture inquiry form (saved to database)
- 📧 Newsletter signup
- 💬 WhatsApp integration (+880 1575-837449)
- 📱 Floating WhatsApp button
- 📊 Admin dashboard at `/admin`

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Database**: PostgreSQL via Drizzle ORM
- **Styling**: Tailwind CSS v4
- **Fonts**: Playfair Display (serif) + Inter (sans-serif)

## Environment Variables

```
DATABASE_URL=your_postgresql_connection_string
```

## Database Setup

Push the schema to your database:

```bash
npx drizzle-kit push
```

## Development

```bash
npm install
npm run dev
```
